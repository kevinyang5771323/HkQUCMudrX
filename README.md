# Python 货物进销管理系统 python853

## 项目概述

本项目是一个基于 Python Flask 框架的货物进销管理系统。该系统集成了采购订单管理、商品信息管理、供应商管理、销售管理、仓库管理、成员管理等功能模块，能够帮助用户高效地完成日常进销存业务处理。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Flask
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 系统运行环境

- Python 3.0 或更高版本
- MySQL 8.0

## 功能模块

- **用户管理**
 - 登录
 - 注册
 - 个人信息修改
- **采购管理**
 - 采购订单管理
- **商品管理**
 - 商品信息管理
- **供应商管理**
 - 供应商信息管理
- **销售管理**
 - 销售记录管理
- **仓库管理**
 - 库存管理

## 系统角色

- 系统管理员
- 采购人员
- 销售人员
- 仓库管理员

## 目录结构

```
项目根目录/
│
├── app/ # Flask 应用目录
│ ├── models/ # 数据库模型
│ ├── views/ # 视图处理逻辑

│ └── static/ # 静态资源
│
├── venv/ # Python 虚拟环境目录
│
└── docs/ # 项目文档
```

## 部署步骤

1. 确保系统中已安装 Python 3.0 或更高版本与 MySQL 8.0。
2. 使用 PyCharm 或命令行创建 Python 虚拟环境，并激活该环境。
3. 安装项目依赖，通常使用 `pip install -r requirements.txt`。
4. 配置数据库连接信息。
5. 运行数据库迁移命令以建立数据库表结构。
6. 启动 Flask 应用。
